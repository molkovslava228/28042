## Описание
Программа "Book Tracker" — это GUI-приложение для учета прочитанных книг. Можно добавлять книги, фильтровать их по жанру и количеству страниц, а также сохранять данные в JSON.

## Инструкция по запуску
1. Скачать или склонировать репозиторий.
2. Установить Python (если еще не установлен).
3. Запустить файл `main.py` командой:
```bash
python main.py
